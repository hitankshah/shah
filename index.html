Valid8 - JavaScript Validation Library

Overview

Valid8 is a lightweight and flexible JavaScript validation library. It helps validate data structures such as strings, numbers, booleans, arrays, and objects with customizable validation rules.

Installation

Install Valid8 from NPM:

npm install valid8

Usage

Import and use the library:

const Validator = require("valid8");

const schema = Validator.string().required().minLength(3);
const result = schema.validate("Hi");
console.log(result); 
// { success: false, errors: ['Minimum length is 3.'] }

Validation Methods

required()

Ensures the value is not null, undefined, or an empty string.

Validator.string().required();

minLength(length)

Ensures a string has at least length characters.

Validator.string().minLength(3);

maxLength(length)

Ensures a string has at most length characters.

Validator.string().maxLength(10);

min(number)

Ensures a number is greater than or equal to number.

Validator.number().min(5);

max(number)

Ensures a number is less than or equal to number.

Validator.number().max(100);

validate(value)

Validates the given value against the schema.

const schema = Validator.string().required().minLength(3);
console.log(schema.validate("Hi")); 
// { success: false, errors: ['Minimum length is 3.'] }
